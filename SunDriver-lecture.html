SunDriver - The lecture

The lecture

  • Duration: 1 semester (in combination with project management (not included here)
  • Level: Bachelor or Master in Engineering, Material Science, Physics, Chemistry
  • Requirements: NO prior knowledge of programming, it starts at zero
  • Final goal: students should be able to write small programs (read data from file, process data, write output to file) and generate output with graphs and HTML pages
The lecture addresses common tasks for the topic of modelling and simulation within the context of engineering. Here, we use the setup of a solar panel system as an example problem and work through it like it would be a real project work. This lecture consists of five parts.
  1. Introduction to programming and the PYTHON language.
    • Where to find information?
    • How to run a program and what are data types?
    • Working with strings - using text encryption as a working example (the Caesar cipher).
  2. How to use functions and modules?
  3. Create simple plots with MathPlotLib.
  4. Built a system model - the Scooter Project.
    • The scooter model
    • The solar model
    • The battery model
    • The system model
  5. Preparing the final results

  • The battery model
    • The battery model is a very simple model. It provides two functions: charging and discharging a battery.
    The following considerations add some more complexity:

    For the function of discharging: You cannot discharge more than what is stored.
    Check, if you want to discharge to much.
    • YES: Discharge all there is / battery will be empty.
    • NO: You can safely discharge the requested amount.

    For the function of charging: You cannot exceed maximum capacity.
    Check, if you want to charge to much.
    • YES: Charge only the amount until you reach maximum capacity.
    • NO: You can safely charge the requested amount.

5. Preparing the final results

*** Your task for the final report ***

Question 1):
What is the minimum size of a PV system in order to drive the scooter for a whole year?
Choose the city of Emden and at least one other city.

Question 2):
For the results of Question1 -> What is the total cost of driving the scooter – calculated per year?
Assume the following prices:
  • Scooter = 350 EUR (incl. battery)
  • Battery = 350 EUR (for PV, size 480 Wh identical to scooter)
  • PV panel = 1 EUR / 1 Watt peak power
The cost (CAPEX) are distributed over 10 years, i.e. divide the sum by 10 years to obtain the driving cost.

How to run the program:
  • You can either use a batch file with the line 'python runsystem.py > output.txt' or
  • Open the Windows PowerShell, navigate to the correct directory and type 'python runsystem.py'.
Steps to be taken:
  1. Open the file 'runsystem.py' in NotePad++
  2. In line 13 set the size of PV system.
  3. In line 20 set the number for the correct city.
  4. Run the program.
  5. Check the HTML pages 'results-energy.html' and 'results-battery.html' for output.
    (Remember to reload the pages, if new output was generated.)
