# Students task #

    (This is for message 1.) Modify your decoding program in a way that it tests for all possible variants of the shift algorithm. Generate text output and decide, which shift was used for encoding.
    This is for message 2.) Modify you program in order to process text which may also contain capital letters and the characters ' . , ? ' (dot, comma, question mark).Hint: For capital letters you may check the functions for strings: string.isupper() or string.islower()
    Modify your program in the way that it produces an error message, if the original text contains characters that are not recognized. Decide, what your program should do in this case.


# Example workflow #

The original message is:
this is an example text for the lecture technology of circular economy!

Your program for decoding should check all possible shifts and create some output.
You can read and understand the message if you found the 'correct' value for the shift.
The key for the solution is to create a loop around your decoding program. 
You can decide to follow the last lecture and put the decoding part into a function.
This function then can be called from within a loop. But there are also other ways to do it.
